##Super Awesome Fish

A simple boilerplate for webpack and react.
This is under a lot of construction right now.

###Step 1
```
clone me the way you clone things in git
```

###Step 2
```
npm install
```

###Step Goldfish
If you want to just build

```
npm run build
```

If you want run dev server

```
npm run dev
```

###Step 4
Eat a sandwich

```
sudo eat sandwich
```
